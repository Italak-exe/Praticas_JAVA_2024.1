return a sum of logratihms with the base X: 

**Retorne a soma do logaritmo de dois valores(A e B) em uma base X qualquer:

**Mudança de base : log natural do valor / log natural da base

1º Com Valor definido{
	- Criar uma função que realiza a mudança de base;

	- Atribuir valores para cada uma;

	- Criar a variavel de soma;

	- Fim.
}
